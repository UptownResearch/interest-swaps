# Interest Swaps

Built from truffle react box (https://github.com/truffle-box/react-box).

Install packages:
cd client && npm install


Run React server:
cd client && npm run start



Run in develop:
```
ganache-cli #(in another window)
truffle migrate --network develop
truffle develop
``
